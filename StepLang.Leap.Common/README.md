# StepLang.Leap.Common

A library of common components for interacting with LEAP APIs.

> LEAP stands for _Library ExchAnge Platform_ and is a package repository for
> STEP libraries.

## Installation

```bash
dotnet add package StepLang.Leap.Common
```
